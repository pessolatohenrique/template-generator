# Template para Node com Nestjs

### Sobre

O objetivo deste projeto é fornecer um template com recursos comuns à todos os projetos em Node.js com Nestjs

### Tecnologias

- Node.js (recomenda-se > 20.0.0)
- Nestjs
- MySQL
- JSON Server

### Instalação do projeto (sem o docker)

Acesse a pasta do projeto e rode o comando para instalar as dependências npm:

    npm install

Execute o projeto:

```
  npm start
```
